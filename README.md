# byte-api-cli

根据.dbml 文件生成 Restful 接口

### Installation

```shell
npm i -g @byte-web-base/byte-api-cli

```

### Github OAuth 登录

登录后才能使用 cli, 方便收集信息以更好的改进脚手架的体验

```shell
byte-api-cli login
```
